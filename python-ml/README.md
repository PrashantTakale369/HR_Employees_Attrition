# Python ML Backend

This folder contains the setup files for the Python-based machine learning backend.

## Setup

**Windows:**
```bash
setup.bat
```

**Linux/Mac:**
```bash
chmod +x setup.sh
./setup.sh
```

## Files

- `requirements.txt` - Python package dependencies
- `setup.bat` - Windows setup script
- `setup.sh` - Linux/Mac setup script
- `data/` - Dataset folder containing IBM HR Employee Attrition data

## Note

The ML model training and API files have been removed from this repository. This folder now only contains the basic setup structure and data.
